About This Document and My Role

Purpose

This document was provided to administrators to assist them in adding their devices to a network so that the application could monitor the health of the devices.

My Role

  1. Received the documentation from the developers
  2. Analyzed the documentation to get a better understanding about the feature
  3. Reviewed if the current structure aligned with the feature
  4. Rewrote and reorganized the content for a better understanding of the feature
  5. Worked with developers when I needed to have a better understanding of what they were trying to say
  6. Completed a first draft and sent it back to developers for comments
  7. Incorporated the developers feedback into the documentation
  8. Repeated this process until the documentation was completed
  9. Sent for one last review and signoff of completed documentation
